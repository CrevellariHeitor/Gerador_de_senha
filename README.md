Este projeto consiste em um gerador de senhas simples desenvolvido em Python, com o objetivo de criar senhas aleatórias e seguras a partir da combinação de letras maiúsculas e minúsculas, números e caracteres especiais. O usuário pode definir o tamanho desejado da senha, e o programa realiza a geração automaticamente. A aplicação é executada via terminal e foi criada com foco na prática de lógica de programação, manipulação de strings e uso de recursos nativos do Python para geração de dados aleatórios. Trata-se de um projeto básico, porém funcional, que pode ser expandido futuramente com novas funcionalidades, como definição de regras mínimas de segurança, geração de múltiplas senhas, interface gráfica ou até mesmo integração com uma API.
